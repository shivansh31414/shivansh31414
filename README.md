### greater_maths — scientific-toolkit (Node.js) + CLI + Plugin System
**What the project does:**  
`scientific-toolkit` is a comprehensive **Node.js scientific computation framework** with a **plugin architecture** and a **CLI**, built for scientists, engineers, and researchers. It supports computational math primitives, numerical methods, domain-specific plugins, and exporting results into publication-friendly LaTeX.

**Core capabilities:**
- **Core Mathematics:** complex numbers, linear algebra (determinant/gaussian elimination), calculus (numerical derivative + integration).
- **Numerical Methods:** Newton-Raphson root finding, ODE solvers (Euler + RK4), flexible numerical integration/differentiation.
- **Vector Calculus:** gradient, divergence, curl.
- **Regression & Fitting:** linear regression (R²), polynomial fitting via normal equations.
- **Engineering modules:** circuit analysis (RLC impedance, Ohm’s law), stoichiometry (molar mass, theoretical yield, reactions).
- **Probability distributions:** normal, poisson, binomial.
- **Advanced:**  
  - **Plugin system** (built-ins include *astrophysics* + *bioinformatics* plugins)  
  - **LaTeX export** for equations, matrices, and results  
  - **CLI interface** for math/probability/unit conversion workflows  
  - **Unit conversion** with dimensional analysis
- **Dev/CI quality:** Jest tests + coverage, CI matrix (Node 16/18/20), ESLint + Prettier, TypeDoc docs, tutorials + examples.

**Examples included (practical use cases):**
- Orbital mechanics (orbital/escape velocity, etc.)
- Bioinformatics (GC content, transcription, reverse complement, Hardy–Weinberg)
- Numerical methods (ODE solving + fitting)
- Engineering analysis (circuits + stoichiometry)

**Stack:**
- Frontend: **React + TailwindCSS**

- DB: **PostgreSQL + Prisma**

**Key features:**
- **MSME signup/login** using **JWT authentication**
- Business profile capturing: business name, sales, expenses, digital presence score
